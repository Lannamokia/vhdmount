# Maimoller HID 系统菜单指南

本文档详细说明 `VHDMounter_Maimoller.exe`（增强版客户端）中 Maimoller HID 操控面板的完整功能与操作方式。

---

## 支持的硬件

增强版客户端自动识别并连接 Maimoller 操控面板。

| 属性 | 值 |
|------|-----|
| Vendor ID | `0x0E8F` |
| Product ID | `0x1224` |
| Report ID | `0x01` |
| 数据包长度 | 7 字节 |

**P1 / P2 检测**：若连接了多个面板，客户端优先选择 P1。

---

## 按钮定义

Maimoller 面板上有 8 个数字按钮和 4 个系统按钮：

| 按钮类型 | 名称 |
|----------|------|
| 数字按钮 | Button 1 ~ Button 8 |
| 系统按钮 | Coin、Service、Test、Select |

---

## 输入路由模式

客户端根据当前界面状态自动切换两种输入模式：

| 模式 | 适用场景 |
|------|----------|
| **导航模式** | 系统菜单、系统信息、网络设置选择页、音频设置等 |
| **数字编辑模式** | 静态 IPv4 地址输入界面 |

---

## 导航模式按键映射

在导航模式下，各按钮对应的功能如下：

| 面板按键 | 对应功能 |
|----------|----------|
| Button 6 | 上 / 上一页 |
| Button 3 | 下 / 下一页 |
| Button 4 | 确认 |
| Button 5 | 返回 / 关闭菜单 |
| Coin 长按 15 秒 | 打开系统菜单 |

> Coin 长按检测：从按下开始计时，持续 15 秒未松开则触发菜单打开。菜单已打开、存在待处理电源操作或当前处于错误状态时，长按请求会被忽略。

---

## 数字编辑模式按键映射

在静态 IPv4 输入界面，输入切换为数字编辑模式：

| 面板按键 | 对应输入 |
|----------|----------|
| Button 1 | 数字 `1` |
| Button 2 | 数字 `2` |
| Button 3 | 数字 `3` |
| Button 4 | 数字 `4` |
| Button 5 | 数字 `5` |
| Button 6 | 数字 `6` |
| Button 7 | 数字 `7` |
| Button 8 | 数字 `8` |
| Test | 数字 `9` |
| Service | 数字 `0` |
| Coin 短按（< 1 秒松开） | 切换到下一个输入段；连续短按三次返回 |
| Coin 长按（>= 1 秒松开） | 保存当前静态 IPv4 配置 |

### IPv4 输入规则

- 每个字段（IP 地址、子网掩码、网关、DNS）分为 **4 段**，每段用 `.` 分隔
- 每段最多输入 **3 位数字**
- 每段数值范围 **0 - 255**
- 子网掩码额外验证：必须是**连续的 IPv4 掩码**
- 必填字段：IP 地址、子网掩码；可选：网关、主要 DNS、备用 DNS

---

## 系统菜单结构

系统菜单包含 4 个主项，分两页显示（每页 2 项）：

| 页码 | 菜单项 | 说明 |
|------|--------|------|
| 第 1 页 | 重启系统 | 卸载当前挂载后立即重启 Windows |
| 第 1 页 | 关闭系统 | 卸载当前挂载后立即关闭 Windows |
| 第 2 页 | 系统信息 | 查看 CPU、内存、独显与磁盘占用情况 |
| 第 2 页 | 系统设置调整 | 进入网络设置和音频设置 |

> 翻页提示：第 1 页底部显示"按 3 可进入下一页"，第 2 页显示"按 6 可返回上一页"。

---

## 系统信息详情

系统信息分为多页，使用 Button 6（上一页）和 Button 3（下一页）切换：

| 页码 | 标题 | 显示内容 |
|------|------|----------|
| 1 | CPU | 占用率、温度 |
| 2 | 内存 | 占用率、已用 / 总量 |
| 3 | 独显 | 名称、占用率、温度、驱动版本、显存 |
| 4+ | 分区 | 每页最多 3 个分区（名称、占用率、已用/总量/剩余） |

> 若未检测到独显，第 3 页显示"未检测到"。

---

## 系统设置调整

### 网络设置

进入后流程如下：

1. **网卡选择**：若检测到多张已连接网卡，每页显示 3 个，Button 6/3 切换，Button 4 确认
2. **模式选择**：
   - **DHCP**：自动获取 IPv4 地址和 DNS。确认后应用，失败会显示错误信息
   - **静态 IPv4**：进入数字编辑器手动配置 IP、子网掩码、网关、主要 DNS、备用 DNS
3. **应用结果**：成功后会刷新网卡状态并显示当前配置

### 音频设置

- 显示系统默认输出设备名称
- 显示当前音量（进度条形式）
- 若未找到可用设备，显示"未找到可用的默认输出设备"

---

## 电源操作安全机制

选择"重启系统"或"关闭系统"后，会进入二次确认页：

- **Button 4（确认）**：立即执行电源操作
  1. 尝试卸载当前 VHD 挂载
  2. 尝试停止 EVHD 挂载进程
  3. 执行系统重启或关机
- **Button 5（返回）**：取消操作，返回系统菜单

> 确认期间新的 Coin 长按请求会被忽略，防止误触发。

---

## 故障排查

| 现象 | 可能原因 | 排查方法 |
|------|----------|----------|
| 面板按键无响应 | HID 设备未识别 | 检查设备管理器中 HID 设备状态 |
| 菜单无法打开 | Coin 按住时间不足 | 确保持续按住 Coin 达 15 秒 |
| 系统信息无数据 | 硬件监控服务初始化失败 | 检查是否以管理员身份运行 |
| 网络设置不生效 | 权限不足 | 必须以管理员身份运行才能修改网络配置 |
| 音频设置无设备 | 音频服务异常 | 检查 Windows 音频服务是否运行 |
